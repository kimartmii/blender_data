Основы интерфейса и навигации в Blender:
Blender использует 3D-пространство (Viewport) как основную рабочую зону. Для навигации используйте мышь: зажмите среднюю кнопку (колесико) для вращения камеры вокруг объекта. Удерживайте Shift + среднюю кнопку для панорамирования (перемещения камеры влево/вправо/вверх/вниз). Прокрутка колесика отвечает за зум. Если вы потеряли объект из виду, выделите его в списке справа (Outliner) и нажмите Numpad . (точку на цифровой клавиатуре), чтобы сфокусировать на нем камеру. Справа находится панель Properties (Свойства), где настраиваются рендер, модификаторы и материалы.
\\
Режимы Object Mode и Edit Mode:
В Blender есть два главных режима. Object Mode (Объектный режим) используется для перемещения, вращения и масштабирования объектов целиком в сцене. Edit Mode (Режим редактирования) служит для изменения самой формы объекта на уровне его геометрии: вершин (vertices), ребер (edges) и граней (faces). Переключение между этими режимами осуществляется клавишей Tab. В Edit Mode используйте клавиши 1, 2, 3 (верхний цифровой ряд) для выбора режима выделения: вершины, ребра или грани.
\\
Базовые трансформации (G, R, S):
Для управления объектами используются горячие клавиши. G (Grab) — перемещение, R (Rotate) — вращение, S (Scale) — масштабирование. Чтобы ограничить действие одной осью, нажмите клавишу оси (X, Y или Z) сразу после клавиши трансформации. Например, G -> Z переместит объект только вверх или вниз. Для точного ввода можно набрать цифры: R -> Z -> 90 повернет объект ровно на 90 градусов вокруг вертикальной оси. Отменить действие можно классическим Ctrl + Z.
\\
Инструменты моделирования:
В Edit Mode часто используются три основных инструмента. Экструдирование (Extrude, клавиша E) выдавливает выделенные грани, создавая новый объем — полезно, если вы моделируете, скажем, обложку и страницы для черной тетради. Фаска (Bevel, Ctrl + B) позволяет скруглить острые углы объекта. Инструмент Loop Cut (Ctrl + R) добавляет новые разрезы геометрии (после нажатия Ctrl+R наведите на объект и покрутите колесико мыши, чтобы увеличить количество разрезов) — это отлично помогает при детализации, например, деревянных ящиков для уровней в CS.
\\
Материалы и текстурирование:
Чтобы придать объекту цвет, перейдите на вкладку Material Properties (иконка красного шарика) и нажмите New. Основной шейдер — Principled BSDF. Настройка Base Color меняет основной цвет объекта. Параметр Roughness отвечает за шероховатость: значение 0 сделает поверхность зеркальной или влажной (как у свежего яблока), а значение 1 — полностью матовой (как бумага или ткань). Параметр Metallic делает материал похожим на металл.
\\
Освещение и рендеринг (Eevee и Cycles):
В Blender есть несколько типов источников света: Point (точка), Sun (солнце, светит бесконечно ровно), Spot (прожектор) и Area (площадь, дает мягкие тени). Для финальной генерации картинки (рендера) используются два движка: Eevee и Cycles. Eevee работает в реальном времени, он очень быстрый и подходит для стилизованной графики. Cycles — это физически корректный движок с трассировкой лучей, он нужен для достижения фотореализма, но требует больше времени на просчет. Для запуска рендера картинки нажмите F12.
